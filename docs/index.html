About LuaFox Runtime

what is the luafox runtime:

the luafox runtime is a Lua Virtual Machine designed to create a way for users to create multi platform programs IN LUA!

but to know realy what it is you need to know the storry behind it

i was inspired by the LuaRT project its a framework for lua on windows. bassicly allows you to create windows applications in lua but, ONLY ON WINDOWS! so yeah.

i wanted to get my stuff to work on linux too

however im going to use it to bring my project to windows

i then thought of how java apps are able to work on windows and linux, so then i learned that it uses somthing called JVM its a virtual machine that runs java code as machine code (I think). you compile you java app and it compiles it to the code that the vm undestands. these apps are callled java executables (".jar" files)

however those apps only work for the version of java it was compiled to (but it runs pretty fast though)

i though what if i can make a simmilar concept but with lua it would be interestingso i did its in alpha state but i think its pretty cool

  • instead of .jar files you have .lfar (which is just an archive)
  • instead of compiling for the vm the vm simply provieds custom apis and methods that adapt to the platform its  running on
  • it has its own package manager aswell called LFPKM (LuaFox (runtime) Package Manager)
  • and all previous versions of apis are kept apps can request old versions of apis this is so older applications work even on newer versions of LuaFox-Runtime without needing to install older versions of it
  • and its open source
